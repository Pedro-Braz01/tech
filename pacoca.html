🔐 Configuração GitHub (segura)

O token fica salvo no servidor Cloudflare Worker como variável de ambiente criptografada — nunca no seu browser.

Passo 1 — Token GitHub:
github.com → Settings → Developer Settings → Fine-grained tokens
→ Repository access: seu repo → Permissions → Contents: Read and write

Passo 2 — Cloudflare Worker:
Já deployou o github-proxy-worker.js? Cole a URL abaixo.
Ainda não? Veja o arquivo github-proxy-worker.js — instruções no topo.
Esta URL fica salva no seu browser. O token GitHub fica NO Worker, nunca aqui.
